# Heather Christie + Vir McCoy — Hybrid Set Technical Rider

## Format
Heather Christie DJ set w/ Vir McCoy live on percussion, flutes, guimbri

## Set length
60–120 min

## Stage footprint
Approx 3m x 2m

## DJ rig (for Heather) — venue provides
- 2x Pioneer CDJ-3000 (CDJ-2000NXS2 minimum)
- 1x Pioneer DJM-900NXS2 or DJM-V10 mixer
- USB-ready
- Booth monitor

## Vocal (Heather) — venue provides
- 1x SM58 or equivalent w/ boom stand
- Light reverb/delay on channel

## Vir McCoy instrument channels — venue provides
- 1x instrument mic (SM57 or e906) for percussion (frame drum / djembe / cajón depending on night)
- 1x DI for guimbri (Gnawa bass lute)
- 1x condenser or clip mic for flutes (can share w/ percussion mic if needed)
- 1x wedge monitor stage left (Vir)
- Clean 1/4" input to mixer for guimbri DI

## Monitors

- Stage wedge for Vir

## What Heather brings
- USBs, backup laptop, vocal FX pedal, headphones
