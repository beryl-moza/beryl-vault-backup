# Heather Christie — Solo DJ Technical Rider

## Artist
Heather Christie (solo DJ set with live vocals)

## Set length
60–180 min (flexible; typical 90-120)

## DJ setup — venue provides
- 2x Pioneer CDJ-3000 (or CDJ-2000NXS2 minimum)
- 1x Pioneer DJM-900NXS2 or DJM-V10 mixer
- USB-ready (she brings own music on USB + backup laptop w/ Rekordbox)
- Booth monitors (minimum 2x 12" wedges or equivalent full-range)

## Vocal setup — venue provides
- 1x dynamic vocal mic (Shure SM58 or equivalent) with boom stand
- Short cable run to mixer (mic input on DJM-V10 or separate small vocal channel)
- Light reverb + delay on vocal channel (can be onboard mixer FX)

## Monitors
- Booth monitors required
- IEM preferred for outdoor venues, not required

## Backline Heather brings
- 2x USBs with set
- MacBook Pro w/ Rekordbox (backup)
- Own headphones
- Vocal FX pedal (small, one 1/4" in + out to mixer)

## Power
- 1x standard 110/220V outlet near booth
- For EU shows: venue provides EU adapter if needed

## Sound check
- 30 min minimum before doors

## Green room / hospitality

- WiFi
- 2x clean towels
